Alinex Developer Page

Information about my current private but public available development projects.

Node Server

Modular server application based on Node.js written mostly in CoffeeScript.

Get more information for each module:

  • General docs containing some development rules and tips for the alinex universe
  • alinex-make a built tool to help while developing
  • alinex-error error handler with code view, source map support...
  • alinex-fs enhancesment of the node filesystem library
  • alinex-util different smaller helpers for string, array and objects
  • alinex-config configuration management
  • alinex-validator validation and sanitize of simple or complex values

In the early development and not usable:

  • alinex-monitor process and system monitoring tool
  • alinex-monitor-sensor sensors for the monitoring
  • alinex-server basic webserver based on express
